# 🎬 ShotPipe - Windows 사용자용 설치 패키지

> **AI 생성 파일을 Shotgrid로 자동 업로드하는 도구**

## 📥 다운로드 받으신 파일들

이 패키지는 Windows 사용자들이 ShotPipe를 쉽게 설치할 수 있도록 준비된 파일들입니다.

---

## 🚀 설치 방법 선택

### 🌐 **방법 1: GitHub 자동 다운로드** (권장)
```
SUPER_EASY_INSTALL_WITH_DOWNLOAD.bat 실행
→ 인터넷에서 최신 코드를 자동으로 가져와서 설치
→ 항상 최신 버전 사용 가능
```

### 📦 **방법 2: 포함된 소스코드 사용**
```
SUPER_EASY_INSTALL_LOCAL.bat 실행  
→ 다운로드받은 패키지의 소스코드로 설치
→ 인터넷 연결 불필요 (오프라인 설치)
```

---

## 🎯 **빠른 시작 (가장 쉬운 방법)**

### 1️⃣ **GitHub 자동 다운로드** ⭐⭐⭐ (권장)
```
1. SUPER_EASY_INSTALL_WITH_DOWNLOAD.bat 우클릭
2. "관리자 권한으로 실행" 선택
```

### 2️⃣ **로컬 소스코드** (오프라인 환경)
```
1. SUPER_EASY_INSTALL_LOCAL.bat 우클릭
2. "관리자 권한으로 실행" 선택
3. y 입력하여 진행
4. 10분 대기 (커피 타임 ☕)
5. 완료! 바탕화면에서 ShotPipe 실행
```

## 📋 파일 설명

### 🎯 **설치 파일들**

| 파일명 | 설명 | 특징 |
|--------|------|------|
| `SUPER_EASY_INSTALL_WITH_DOWNLOAD.bat` | **GitHub 자동 다운로드 설치** | 최신 버전, 인터넷 필요 |
| `SUPER_EASY_INSTALL_LOCAL.bat` | **로컬 소스코드 설치** | 오프라인 가능, 빠름 |
| `설치방법_읽어주세요.txt` | **한국어 설치 안내** | 간단 설명 |

### 🔧 **문제 해결 도구들**

| 파일명 | 설명 | 언제 사용? |
|--------|------|-----------|
| `windows_defender_fix.bat` | Windows Defender 차단 해결 | "바이러스 차단" 메시지 시 |
| `install_exiftool.bat` | ExifTool 수동 설치 | 메타데이터 처리 오류 시 |

### 🏗️ **개발자용 빌드 도구들**

| 파일명 | 설명 | 대상 |
|--------|------|------|
| `build_all_distributions.bat` | 전체 배포판 빌드 | 개발자/고급 사용자 |
| `EASY_WINDOWS_BUILD.bat` | 간단 빌드 도구 | 개발자/고급 사용자 |

### 📂 **소스코드**

| 폴더/파일 | 설명 |
|-----------|------|
| `source/` | ShotPipe 전체 소스코드 |
| `source/shotpipe/` | 메인 모듈 |
| `source/vendor/` | ExifTool 등 의존성 |

## 🆘 설치 중 문제가 생기면?

### ❌ **"바이러스 감지" 경고**
```
해결법:
1. "추가 정보" 클릭 → "실행" 버튼 클릭
또는
2. windows_defender_fix.bat 실행 (관리자 권한)
```

### ❌ **"source 폴더를 찾을 수 없습니다" 오류**
```
해결법:
로컬 설치 시 source 폴더가 필요합니다.
→ SUPER_EASY_INSTALL_WITH_DOWNLOAD.bat 사용하거나
→ 전체 패키지를 다시 다운로드하세요.
```

### ❌ **"GitHub 다운로드 실패" 오류**
```
해결법:
1. 인터넷 연결 확인
2. 방화벽 설정 확인  
3. SUPER_EASY_INSTALL_LOCAL.bat 사용 (오프라인)
```

### ❌ **"권한 거부" 오류**
```
해결법:
배치 파일을 우클릭 → "관리자 권한으로 실행"
```

## 💡 사용 순서

### 📥 **1단계: 설치**
```
원하는 설치 방법 선택
→ .bat 파일 우클릭 → 관리자 권한으로 실행
→ 대기 (커피 타임 ☕)
→ 설치 완료!
```

### 🎬 **2단계: 첫 실행**
```
바탕화면의 "ShotPipe" 아이콘 더블클릭
→ 환영 마법사 진행 (3분)
→ 작업 폴더 및 Shotgrid 연결 설정
```

### 📁 **3단계: 파일 처리**
```
1. AI 생성 파일들을 한 폴더에 모으기
2. ShotPipe에서 "파일 처리" 탭 선택
3. 폴더 선택 후 "처리 시작"
4. Shotgrid 업로드까지 자동 완료!
```

## 🔧 시스템 요구사항

### 최소 사양
- **OS**: Windows 10 이상 (64bit)

- **RAM**: 8GB 이상
