=============== Shotgrid API 키 설정 및 테스트 ===============

1. Shotgrid 웹사이트에서 API 키 얻기:
   - https://lennon.shotgunstudio.com에 로그인 (관리자 계정 필요)
   - 우측 상단 아바타 > 관리자 > 스크립트로 이동
   - 'AX_shotPipe' 스크립트 찾기 또는 새로 생성
   - API 키 재설정 또는 확인 (새로 생성 시 키가 표시됨)

2. API 키를 .env 파일에 입력:
   - .env 파일 열기: /Users/onset/AX_pipe/ShotPipe/.env
   - SHOTGRID_API_KEY= 다음에 새 API 키 붙여넣기
   - 파일 저장

3. API 키 테스트:
   - 터미널에서 다음 명령어 실행:
     $ cd /Users/onset/AX_pipe/ShotPipe
     $ python test_api_key.py

4. 애플리케이션 다시 실행:
   - 터미널에서 다음 명령어 실행:
     $ cd /Users/onset/AX_pipe/ShotPipe
     $ python main.py

======= 디버깅 팁 =======

- API 키는 대소문자를 구분합니다. 정확히 복사하세요.
- 스크립트 이름도 대소문자를 구분합니다.
- Shotgrid에서 스크립트가 '활성' 상태인지 확인하세요.
- 스크립트에 충분한 권한이 있는지 확인하세요 (관리자 권한 추천).

자세한 내용은 shotgrid_setup_guide.md 파일을 참조하세요.
